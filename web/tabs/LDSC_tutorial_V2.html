Stratified LD Score Regression (LDSC)

A beginner-friendly guide to understanding how LDSC partitions heritability across functional genomic annotations

What is Stratified LD Score Regression?

🧬

Understanding Heritability

Heritability measures how much of the variation in a trait (like height or disease risk) is due to genetic differences between people. LDSC helps us understand which parts of the genome contribute to this heritability.

The Problem with Standard GWAS

Standard Genome-Wide Association Studies (GWAS) identify individual genetic variants associated with traits. However, they don't tell us:

  • Which functional categories of the genome are most important
  • How heritability is distributed across different genomic annotations
  • Whether certain cell types or tissues play a bigger role

Stratified LDSC extends basic LDSC by partitioning heritability across different functional categories (annotations) in the genome, allowing us to understand which genomic features contribute most to trait heritability.

How Stratified LDSC Works

1

Linkage Disequilibrium (LD)

LD refers to the non-random association of alleles at different loci. In simple terms, genetic variants that are close together on a chromosome tend to be inherited together.

LD Score measures how much a SNP is correlated with other SNPs in the genome.

2

Stratification by Annotations

We divide the genome into different functional categories (annotations), such as:

  • Protein-coding regions
  • Regulatory elements (promoters, enhancers)
  • Conserved regions
  • Cell-type-specific chromatin marks
3

Regression Model

Stratified LDSC uses a regression model where the GWAS χ² statistics are regressed against LD scores from each annotation category:

E[χ²] ≈ N * Σ (τ_c * ℓ(c)) + a + 1

Where:

  • N: Sample size
  • τ_c: Per-SNP heritability for annotation c
  • ℓ(c): LD Score for annotation c
  • a: Intercept (accounts for confounding)

Key Insight

If an annotation is enriched for heritability, SNPs in that annotation will have higher χ² statistics than expected based on their LD scores alone. This allows us to quantify how much each annotation contributes to trait heritability.

Stratified LDSC Workflow in CONSILIENCE-GWAS

🚀 Automated S-LDSC Pipeline

CONSILIENCE-GWAS implements a complete stratified LDSC workflow that automatically processes your inputs and generates comprehensive enrichment analysis with built-in quality control.

Step 1: Input Preparation & Quality Control

1

GWAS Summary Statistics

Upload standard GWAS output files containing SNP information, effect sizes, standard errors, and p-values. CONSILIENCE-GWAS automatically performs quality control including:

  • Filtering ambiguous SNPs (A/T, G/C)
  • Restricting to HapMap3 variants for improved data quality
  • Validating file format and required columns
2

Gene Set Selection

Choose from thousands of curated gene sets via GeneWeaver integration or upload custom gene lists. Gene sets are automatically mapped to human orthologs using biomaRt.

Step 2: Annotation Creation & LD Score Computation

3

Gene-to-SNP Mapping

Gene sets are converted to genomic annotations using flexible window sizes around each gene:

10-100 kb upstream/downstream: Captures promoter regions and nearby regulatory elements
User-configurable: Adjust window size based on trait biology and regulatory landscape

This approach captures both coding regions and regulatory elements associated with each gene.

4

LD Score Computation

Stratified LD scores are calculated for each annotation category using:

  • 1000 Genomes Phase 3 European ancestry reference panel
  • 53 baseline functional annotations for comprehensive control
  • HapMap3 SNPs with MAF > 5% and imputation INFO ≥ 0.9

Step 3: Regression Modeling & Enrichment Analysis

5

Stratified LDSC Regression

The model is fitted using S-LDSC version 1.0.1 with all annotations included simultaneously:

E[χ²] = N * Σ(τ_c * ℓ(j,c)) + a + 1

Where τ_c represents the contribution of annotation c to per-SNP heritability, controlling for all other annotations.

6

Three Analysis Models

CONSILIENCE-GWAS automatically runs three complementary models:

Model Purpose Output
Joint Model Partitions heritability into unique and overlapping components Unique, intersection, and background contributions
Union Model Tests combined effect of all gene sets Cumulative enrichment across all sets
Individual Model Tests each gene set separately Set-specific enrichment estimates

Step 4: Results Interpretation & Output

7

Enrichment Statistics

For each annotation, CONSILIENCE-GWAS calculates:

📈 Enrichment Coefficient

τ = (h²_c/M_c) / (h²_total/M_total)
Measures how concentrated heritability is in the annotation

📊 Proportion of h²

Percentage of total SNP heritability explained by the annotation

🎯 P-values

Statistical significance with FDR correction (q < 0.05)

8

Multiple Testing Correction

All results include comprehensive multiple testing protection:

  • False Discovery Rate (FDR): q < 0.05 threshold for significance
  • Bonferroni correction: For cell-type analyses (P < 0.05/10 = 0.005)
  • Standard errors: Confidence intervals for all estimates

  • Automated quality control: HapMap3 filtering and data validation

Applications in CONSILIENCE-GWAS

1. Cross-Species Enrichment Analysis

Test whether gene sets from model organisms (mouse, rat) show significant enrichment in human GWAS, revealing evolutionarily conserved mechanisms.

2. Tissue and Cell-Type Specificity

Identify which tissues and cell types are most relevant for your trait using pre-computed annotations from GTEx, Roadmap Epigenomics, and specialized datasets.

3. Functional Category Partitioning

Understand how heritability distributes across different genomic features, even for highly polygenic traits.

4. Gene Set Prioritization

Rank candidate gene sets by their enrichment significance to prioritize biological pathways for further investigation.

Example from our paper: Using CONSILIENCE-GWAS, we discovered that binge drinking gene sets from mouse models showed significant enrichment in human alcohol consumption GWAS (enrichment = 1.41, FDR < 0.05), particularly in striatal-prefrontal-ventral tegmental regions.
